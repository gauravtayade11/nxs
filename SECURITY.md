# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 2.x     | Yes       |
| < 2.0   | No        |

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Report security issues by emailing **security@nextsight.dev** or via [GitHub private vulnerability reporting](https://github.com/gauravtayade11/nxs/security/advisories/new).

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

You will receive a response within **48 hours**. If the issue is confirmed, a patch will be released as soon as possible.

## Scope

- `nxs` CLI binary and all commands
- `@nextsight/nxs-cli` npm package
- GitHub Actions workflows in this repo

## Out of Scope

- Issues in third-party dependencies (report directly to those projects)
- Social engineering attacks
